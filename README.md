# vue-h5
vue-h5是一款基于vue+vant的移动端且开箱即用的前端集成式、优化式框架。

#### 主要技术栈
+ 前端MVVM框架：Vue.JS 2.6.11
+ 路由：Vue-Router 3.2.0
+ 状态管理：VueX 3.5.1
+ 数据交互：Axios 0.19.2
+ UI框架：Vant 2.9.0

#### 安装
```
# 克隆项目
git clone https://github.com/zijunwork/vue-h5.git
# 进入项目目录 vue-h5
cd vue-h5
# 安装依赖
yarn add / npm i
# 本地开发启动项目
npm run serve
```
